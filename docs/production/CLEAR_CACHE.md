# 🧹 清除缓存指南

## 📋 缓存类型

在Zeabur部署中，可能需要清除以下几种缓存：

1. **浏览器缓存** - 浏览器存储的页面缓存
2. **Zeabur构建缓存** - Zeabur的构建系统缓存
3. **应用缓存** - 应用内部的内存缓存

## 🌐 方法1：清除浏览器缓存

### Chrome/Edge浏览器

1. **按 `Ctrl + Shift + Delete`**
2. 选择 **"缓存的图片和文件"**
3. 时间范围选择 **"全部时间"**
4. 点击 **"清除数据"**

### 或使用硬刷新

- **Windows**: `Ctrl + F5` 或 `Ctrl + Shift + R`
- **Mac**: `Cmd + Shift + R`

### 或使用无痕模式

- **Windows**: `Ctrl + Shift + N`
- **Mac**: `Cmd + Shift + N`

## 🔧 方法2：清除Zeabur构建缓存

### 在Zeabur控制台

1. **访问**: https://zeabur.com
2. **找到项目**: my-telegram-bot33
3. **点击应用服务**
4. **找到 "Settings" 或 "设置"**
5. **查找 "Clear Build Cache" 或 "清除构建缓存"**
6. **点击清除**

### 或重新部署

重新部署会自动清除构建缓存：

1. **在服务页面**
2. **找到 "Redeploy" 或 "重新部署" 按钮**
3. **点击重新部署**
4. **等待部署完成**

## 💾 方法3：清除应用缓存

应用使用内存缓存，重启服务会自动清除。

### 重启服务

1. **在Zeabur控制台**
2. **找到应用服务**
3. **点击 "Restart" 或 "重启" 按钮**
4. **等待服务重启（1-2分钟）**

### 或通过API清除（如果实现了）

如果应用实现了缓存清除API，可以访问：

```
POST https://my-telegram-bot33.zeabur.app/admin/cache/clear
```

## 🔍 检查缓存状态

### 查看应用缓存

如果应用有缓存监控端点，可以访问：

```
GET https://my-telegram-bot33.zeabur.app/admin/cache/status
```

### 查看应用日志

在服务日志中，查找缓存相关的信息。

## 📋 清除缓存检查清单

- [ ] 清除浏览器缓存（`Ctrl + Shift + Delete`）
- [ ] 清除Zeabur构建缓存（在服务设置中）
- [ ] 重启服务（清除应用内存缓存）
- [ ] 使用硬刷新访问页面（`Ctrl + F5`）

## 🎯 推荐操作流程

### 如果遇到502错误或页面不更新

1. **清除浏览器缓存**
   - 按 `Ctrl + Shift + Delete`
   - 清除缓存

2. **硬刷新页面**
   - 按 `Ctrl + F5`

3. **如果还是不行，重启服务**
   - 在Zeabur控制台重启服务

4. **等待服务完全启动后，再次访问**

## 🆘 常见问题

### Q1: 清除缓存后还是502？

**可能原因：**
- 服务本身有问题
- 端口配置问题
- 环境变量未配置

**解决：**
1. 查看服务日志
2. 检查服务状态
3. 检查环境变量配置

### Q2: 如何清除特定类型的缓存？

**浏览器缓存：**
- 使用 `Ctrl + Shift + Delete`

**构建缓存：**
- 在Zeabur控制台清除
- 或重新部署

**应用缓存：**
- 重启服务
