# 🖥️ 终端使用基础教程（小白版）

## 📍 您当前的状态

您已经连接到Zeabur容器终端，看到类似这样的提示符：
```
root@service-xxx:/app#
```

这个 `#` 符号后面的位置就是您可以输入命令的地方。

## 🎯 如何输入命令

### 步骤1：找到输入位置

在终端窗口中，您会看到：
```
root@service-xxx:/app# █
```

这个闪烁的光标（`█`）就是您可以输入的地方。

### 步骤2：输入命令

**方法1：直接输入**
1. 点击终端窗口（确保光标在终端内）
2. 直接输入命令，例如：`echo $DATABASE_URL`
3. 按 `Enter` 键执行

**方法2：复制粘贴（推荐）**
1. 复制命令（选中命令文本，按 `Ctrl+C` 或右键复制）
2. 在终端中点击右键，选择"粘贴"
3. 或按 `Ctrl+V` 粘贴
4. 按 `Enter` 键执行

### 步骤3：查看结果

命令执行后，结果会显示在终端中。

## 📝 第一个命令：检查DATABASE_URL

### 完整操作步骤

1. **复制这个命令：**
   ```
   echo $DATABASE_URL
   ```

2. **在终端中粘贴：**
   - 点击终端窗口
   - 右键点击，选择"粘贴"
   - 或按 `Ctrl+V`

3. **按 Enter 键执行**

4. **查看结果：**
   - 如果看到类似 `postgresql://postgres:xxx@xxx.zeabur.com:5432/postgres`，说明已设置 ✅
   - 如果什么都没有（空白），说明未设置 ❌

## 🎓 常用命令详解

### 命令1：检查环境变量

**命令：**
```bash
echo $DATABASE_URL
```

**说明：**
- `echo` = 显示/打印
- `$DATABASE_URL` = 环境变量的值
- 这个命令会显示DATABASE_URL的值

**如何输入：**
1. 复制上面的命令
2. 在终端中粘贴
3. 按 Enter

### 命令2：检查所有环境变量

**命令：**
```bash
env | grep -E '(DATABASE|FACEBOOK|OPENAI|TELEGRAM|SECRET)'
```

**说明：**
- 显示所有包含这些关键词的环境变量
- 可以看到哪些变量已设置，哪些未设置

**如何输入：**
1. 复制上面的命令
2. 在终端中粘贴
3. 按 Enter

### 命令3：测试数据库连接

**命令：**
```bash
python -c "from src.core.database.connection import engine; from sqlalchemy import text; conn = engine.connect(); print('✅ 数据库连接成功'); conn.close()"
```

**说明：**
- 测试应用是否能连接到数据库
- 如果成功，会显示 "✅ 数据库连接成功"
- 如果失败，会显示错误信息

**如何输入：**
1. 复制上面的命令（注意：要复制完整的一行）
2. 在终端中粘贴
3. 按 Enter

### 命令4：运行数据库迁移

**命令：**
```bash
alembic upgrade head
```

**说明：**
- 创建数据库表结构
- 这是必需的步骤

**如何输入：**
1. 复制上面的命令
2. 在终端中粘贴
3. 按 Enter

## ⚠️ 常见问题

### Q1: 命令输入后没有反应？

**可能原因：**
- 命令还在执行中（等待几秒）
- 命令执行完成但没有输出（这是正常的）

**解决方法：**
- 等待几秒钟
- 如果看到新的提示符（`#`），说明命令已执行完成

### Q2: 出现错误信息？

**不要担心！** 错误信息可以帮助我们找到问题。

**请把错误信息复制给我，我会帮您解决。**

### Q3: 如何复制终端中的文本？

**方法：**
1. 选中要复制的文本（用鼠标拖选）
2. 按 `Ctrl+C` 复制
3. 或右键点击，选择"复制"

### Q4: 命令输入错误怎么办？

**方法：**
- 按 `Ctrl+C` 取消当前命令
- 重新输入正确的命令

### Q5: 如何清屏？

**命令：**
```bash
clear
```

**或按：**
- `Ctrl+L`
