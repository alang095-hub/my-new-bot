# 📝 Facebook 产品信息填写指南

## 🎯 产品信息填写

在 Facebook Developers 中添加 Messenger 产品时，可能需要填写产品信息。

---

## 📋 常见需要填写的信息

### 1. 产品名称（Product Name）

**建议填写：**
```
iPhone 贷款客服机器人
```
或
```
多平台客服自动化系统
```
或
```
Customer Service Bot
```

**说明：**
- 可以是中文或英文
- 简洁明了，描述产品功能
- 可以是你的业务名称

---

### 2. 产品描述（Product Description）

**建议填写（中文）：**
```
用于 iPhone 贷款业务的自动化客服系统，支持 Facebook Messenger 自动回复客户咨询。
```

**或（英文）：**
```
Automated customer service system for iPhone loan business, supporting Facebook Messenger auto-reply for customer inquiries.
```

**说明：**
- 简要描述产品用途
- 说明主要功能
- 可以是中文或英文

---

### 3. 用例类型（Use Case）

如果要求选择用例类型，通常选择：

- **"客户服务"** 或 **"Customer Service"**
- **"消息机器人"** 或 **"Messaging Bot"**
- **"自动化回复"** 或 **"Auto Reply"**

---

### 4. 业务类型（Business Type）

如果要求选择业务类型，通常选择：

- **"金融服务"** 或 **"Financial Services"**
- **"贷款服务"** 或 **"Lending Services"**
- **"其他"** 或 **"Other"**

---

## ⚠️ 重要提示

### 1. 如果字段是可选的

- **可以留空**（如果允许）
- **或填写简单的描述**

### 2. 如果字段是必填的

- **必须填写**
- **可以使用上面的模板**

### 3. 产品名称要求

- 通常没有特殊要求
- 可以是中文或英文
- 简洁明了即可

## 🔍 具体场景

### 场景 1：添加 Messenger 产品时

如果是在"添加产品"页面：

1. **找到 "Messenger" 产品**
2. **点击 "设置" 或 "Set Up"**
3. **如果要求填写产品信息，使用上面的模板**

### 场景 2：配置用例时

如果是在"定制用例"页面：

1. **勾选 "Messenger from Meta"**
2. **如果弹出表单要求填写信息，使用上面的模板**

### 场景 3：其他情况

如果是在其他页面要求填写产品信息：

- **产品名称**：填写你的业务名称或产品名称
- **产品描述**：简要描述产品功能
- **其他字段**：根据实际情况填写
