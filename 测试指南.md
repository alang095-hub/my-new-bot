# 🚀 快速测试指南（小白专用）

## 方法1：最简单的测试（推荐新手）

### 步骤1：安装Python依赖
在项目文件夹打开PowerShell，运行：
```powershell
pip install -r requirements.txt
```

### 步骤2：检查配置文件
确保 `.env` 文件存在并配置了基本信息，最少需要：
```
DATABASE_URL=sqlite:///facebook_customer_service.db
DEBUG=true
HOST=0.0.0.0
PORT=8000
```

### 步骤3：启动服务
```powershell
python -m src.main
```

### 步骤4：打开浏览器测试
打开浏览器访问：
- http://localhost:8000 （查看首页）
- http://localhost:8000/docs （查看API文档）
- http://localhost:8000/health （查看健康状态）

**看到JSON响应就说明服务启动成功！**

## 🔍 常见问题

### 问题1：启动失败，提示缺少模块
**解决**：重新安装依赖
```powershell
pip install -r requirements.txt
```

### 问题2：端口被占用（Port 8000 already in use）
**解决**：更换端口或关闭占用8000端口的程序
```powershell
# 方法1：使用其他端口
$env:PORT="8001"
python -m src.main

# 方法2：查找并关闭占用端口的程序
netstat -ano | findstr :8000
```

### 问题3：数据库连接错误
**解决**：使用SQLite（最简单）
在 `.env` 文件中设置：
```
DATABASE_URL=sqlite:///facebook_customer_service.db
```
