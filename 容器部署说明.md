## 🔍 您的Dockerfile做了什么？

```dockerfile
FROM python:3.9-slim          # 使用Python 3.9
WORKDIR /app                   # 设置工作目录
COPY requirements.txt .        # 复制依赖文件
RUN pip install ...            # 安装所有依赖
COPY . .                       # 复制应用代码
EXPOSE 8000                    # 暴露端口
CMD uvicorn ...                # 启动FastAPI应用
```

**Zeabur会自动执行这些步骤！**
